# Inferna
---
Inferna is a moderation and security bot I am making for a private discord server. 

Inferna will contain features such as:
- Advanced moderation
- Advanced security
- User Admin & Logging
- Auto-kick (if an account is created 3 days prior to joining - mainly used as an anti-alt system). 

Feel free to use the code however you want, make changes, add new content, and make improvements.

This readme will be updated regularly in accordance to the progress that has been made.
